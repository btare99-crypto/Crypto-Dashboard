# Crypto Dashboard

A responsive **crypto dashboard-style web application** built using **HTML, CSS, and JavaScript**.
The project simulates a crypto platform where users can create an account, log in, manage their profile, and customize their interface.

In the first phase of the project, the focus is on **user authentication, profile management, and UI customization**, while the crypto-related functionality will be implemented in later stages.

# Features

* User registration and login (simulated with **localStorage**)
* Personal user profile
* Edit profile information
* Change **username**
* Change **email**
* Change **full name**
* Update **bio**
* Update **password**
* **Dark mode / Light mode toggle**
* Modern crypto-style dashboard UI
* Responsive layout for desktop and mobile

# Pages

## Login

User authentication page
Includes validation and error handling

## Register

Create a new user account
Password confirmation and input validation

## Dashboard

Main dashboard interface after login

## Settings

User profile management page where users can:

* Edit username
* Edit email
* Edit full name
* Update bio
* Change password
* Toggle **dark mode**

# Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (ES6+)**
* **localStorage** for temporary data storage
* **Font Awesome** for icons

# How to Run

1. Clone or download the project
2. Open the project folder
3. Open **index.html** in a web browser

or

Use **VS Code** with the **Live Server** extension.

# Current Limitations

* Authentication is **frontend-only**
* Data is stored in **localStorage**
* No **backend server**
* No **real database**
* No **crypto API integration yet**
